# Changelog

所有重要的更改都将记录在此文件中。

## [1.0.0] - 2026-03-09

### 新增
- 初始版本发布
- 支持多实例 Docker 部署
- 实例生成脚本 (`generate-instance.sh`)
- 实例管理脚本 (`manage-instances.sh`)
- 预定义角色模板（管家、助手、专家、创意）
- 独立端口分配机制
- 个性化配置文件生成
- 完整的文档和快速开始指南

### 特性
- 每个实例完全隔离运行
- 独立的性格、记忆、技能配置
- 支持多种消息渠道（飞书、Telegram、Discord等）
- 灵活的角色自定义
- 备份和恢复功能
